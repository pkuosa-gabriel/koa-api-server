# koa-api-server

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

A simple RESTful API server based on [koa.js](https://github.com/koajs/koa).

## Tech Stack

- Typescript
- Koa.js
- PostgreSQL
- Jest
